Using the Fringe Control and tracing fringes

You can select points on a fringe using the left mouse button. The right mouse button will delete a point it is close to. If you drag the mouse with the left button down then it will place points along the path. If you drag the mouse with the right button down then it will erase points along the path.

Fringe count /RMS Fringe Fit
Displays the fringe count and after a Calculate Zernikes displays the RMS error of the fringe fit in waves. It usually possible to get an RMS of the fringe fit of .03 or less. If not recheck the fringe points on each fringe. You can use the outlier command to see points that may be causing the problem.

You can also use Short cut keys to help speed up fringe tracing.

The f key:
The f keyboard key is especially useful. Pressing when there is only one point on the fringe it will auto trace the whole fringe. Sometimes you may have to select the dark or light radio button on the fringe before you press the f key.

Pressing it when there are multiple points on the fringe will adjust all the points to the center of the fringe. This is called optomizing a fringe.

Auto fringe finding of all fringes.

To Search for all fringes at once:
Note: works only when no fringes have been defined. If there are fringes press the delete all fringes before trying using the auto select all fringes technique.
  1. Select point on first fringe
  2. Select second point so that blue line is across the whole interferogram and perpendicular to the middle of the fringes.
  3. Select find all fringe buttons.

It is not always perfect and you may have some editing to do. But it does speed up the process a lot. You can select light, dark or both fringes for this.

Inserting fringes between two other fringes

Use the v key or the e key to change the selected fringe (highlighted in blue.) Insert will add a new fringe in front of the currently selected fringe. It will add only one point (a blue '+'.) You can then either press the f key to have the computer find the rest of the fringe or you can trace it yourself with the left mouse button.

Append to current fringe

To append to current fringe hold down left shift key when left clicking to add a point. It will try to find a fringe segment and add it to the current fringe. After this operation type the f key or click on auto trace/optomize
