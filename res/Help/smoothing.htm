Smoothing the analysis

FFT analysis can pick up a lot of noise from the interferometer system. That noise produces spurious spikes and dips in the surface. These need to be smoothed out. To start with it is best to take several images at several tilt settings. Then rotate the mirror 45 deg and do the same all over again. After analysis each wavemap from each rotation can be averaged and then counter rotated using the Rotate button on the surface analysis controls. Then average all the counter rotated averages. There will still be some bumps left over and at that point you will need to use the Gaussian Blur function on the surface filter.

Test Stand astig removal

DFTFringe provides a feature to help automate the analysis and counter rotation of mirrors rotated in interferograms. That feature also removes both test stand induced astig and bath beam seperation astig from the analysis while leaving the true mirror astig. It can be accessed from the Tools menu. The user should take interferograms at pairs of rotation angles that are 90 degrees apart. For example rotate the mirror (0,90) and (45,135) degrees. Take several interferograms at each rotation. Then use the test stand astig removal tool to complete the analysis.
